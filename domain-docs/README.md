# Domain Docs

Questa directory contiene esclusivamente documentazione di dominio:

- PRD
- prompt operativi
- execution plan
- progress tracking

Il codice runtime dei moduli non vive qui.

Riferimenti canonici:

- backend: `backend/app/modules/<modulo>/`
- frontend: `frontend/src/app/<modulo>/`

Esempio:

- documentazione Anagrafica: `domain-docs/anagrafica/docs/`
- backend Anagrafica: `backend/app/modules/anagrafica/`
- frontend Anagrafica: `frontend/src/app/anagrafica/`
